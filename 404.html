Error 404: Page Not Found

\documentclass[12pt]{amsart}
\theoremstyle{plain}
\newtheorem{Theorem}{thm}
\begin{document}

\begin{thm}[Theorem 404]
The page you are looking for does not exist.
\end{thm}

\begin{proof}
By way of contradiction, assume that the
page you seek did exist. Since the page
exists, it must have a valid url. But the
url you have entered is invalid. This is
a contradiction, which completes the proof.
\end{proof}

We recommend using your browser's back
button, checking the url you want and
trying again. If that fails, contact the
page owner.

\end{document}
